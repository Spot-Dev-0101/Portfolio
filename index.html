cinematic

My favorite game is Counter-Strike. I've put thousands of hours into getting better at every aspect and I will continue to.

I use Unreal Engine 5's tools like Path-Tracing, Nanite, Luman, Cine Cameras, Niagra, and many more to bring the weapons to life.

VRSigns

Unity

C#

VR

For my University final project I wanted to make something that would be useful to people in a comminity I'm active in. During Covid I spent a lot of time in VR, specifically VRChat. VRChat is an amazing place for people from all over the world to meet and interact. However, some users with hearing disabilities find this very difficault so the hearing impared community has come up with their own VR sign language. It allows them to communicate, but it is hard to learn because there aren't many resources available and you can only learn from someone that already knows it.

So for my project I created the prototype for a VR tool to teach sign language. The user simples follows the instructions and an algorithm deturmins if they've done the sign successfully.

SkiTracker App

Android

Java

Every year my family and I got skiing, but my mum stays at the bottom of the mountains while we're skiing, because of this for a University project I created an Android app that allows her to see our location in real-time on a map while we're on the mountain.

The app lets you see the location, speed, and altitude of your friends on a live map. It also has a friend code system so you can easily share a 6 digit number with them and they can add you to their list.

VRPiano

Unity

C#

VR

For a University project I was tasked with creating a VR learning experience. Using the hand tracking offered by the Meta Quest 2 I created a simple app that lets you interact with a virtual piano.

This app has two modes. The first is free play that lets you play anything you want for unlimited time.

The second is the learning mode where the user selects a song and the user follows the notes to play the song.
